Saqwel Remote Administration Tool help page

Saqwel Remote Adnministration Tool (SRAT) is an application for computers remote management. It allows to manage Windows client and server operating systems.

This help page try to explain how to use this application.

Official site: www.saqwel.ru.

What is inside

SRAT written in Visual Basic. It uses WMI to get information from remote computers and to set parameters to remote computers. Certainly, you need permissions to connect to remote computers in your network. Firewall has to be set to allow connection through WMI (range of ports you can find on microsoft.com) and VNC (5900 or 5800) ports.

First launch

When you run Saqwel Remote Administration Tool, it tries to find domains and workgroups of your network and computers of domain or workgroup and create list of found computers. When you see list of computers you can click each computer name then click button under main menu and get a result in right panel.
